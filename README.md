2do_proyecto_de_simulacion
==========================

Proyecto de simulacion y prolog

Para correr un agente llame al predicado programprogram(Agente,N,M,PS,PO,PN,T):
Agente: Es el agente que se va a usar en la simulacion (agent1 ó agent2).
N:Largo del tablero.
M:Ancho del tablero.
PS:Por ciento de casillas sucias.
PO:Por ciento de obstaculos.
PN:Por ciento de niños.
T:El valor del tiempo de unidades de cambio
